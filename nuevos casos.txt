Como te comente, he diseñado un juego que consiste en que cada dia, voy a mostrarle al usuario un caso policial, que debe resolver, con un sistema de puntos y rankins para competir con otros usuarios.

Observaciones a tener en cuenta:
Que sea un caso nuevo, que nunca me hayas dicho.
Que en las declaraciones, los personajes puedan nombrar a otros para despirtar o descartar y sean en primera persona
Se pueden incluir horas, tanto en la descripcion como en las declaraciones, para descartar o incriminar personajes
Tienen que ser dos pistas, la que tenga orden 1, tiene que ser una pequeña ayuda, pero la pista con orden 2, tiene que ser una pista que casi ayude a resolver el caso
Lo suyo es que el usuario pueda resolver el caso sin ver las pistas
Que la dificultad sea un numero del 1 al 5, que se relacione con la longitud de la descripcion, si va a ser corta que sea dificultad 1, pero si la haces muy larga que sea dificultad 5, ve alternando, por ejemplo dificultad 1 500 caracteres, dificultad 5 1000 caracteres
La descripcion del caso y las declaraciones deben tener minimo 500 caracteres
Que sean inmersivos

Genera 10 casos policiales pero para hacer insert directamente en las siguentes tablas / campos de supabase:
Caso( "id" ( Debe ser el siguinte id ), "Titulo", "Descripcion", "Resumen", "Dificultad", "FechaDisponible" )
Pista ( "case_id" ( id del caso que se hace el insert ), "Orden", "Descripcion", "Titulo" )
Sospechoso( "case_id" ( id del caso que se hace el insert ), "Nombre", "Descripcion", "Asesino", "Resolucion" )

Que tenga el siguiente formato de insert:
DO $$
DECLARE
  nuevo_id INTEGER;
  nueva_fecha DATE;
BEGIN

  INTO nuevo_id, nueva_fecha 
  FROM "Caso";

  nuevo_id := nuevo_id + 1;
  nueva_fecha := nueva_fecha + INTERVAL '1 day';

  INSERT INTO "Caso" ("id", "Titulo", "Descripcion", "Resumen", "Dificultad", "FechaDisponible")
  VALUES (
    nuevo_id,
    '',
    '',
    '',
    1,
    nueva_fecha
  );

  INSERT INTO "Pista" ("case_id", "Orden", "Descripcion", "Titulo") VALUES
  (nuevo_id, 1, '', ''),
  (nuevo_id, 2, '', '');

  INSERT INTO "Sospechoso" ("case_id", "Nombre", "Descripcion", "Asesino", "Resolucion") VALUES
  (nuevo_id, '', 
   '', 
   false,
   ''),
  
  (nuevo_id, '', 
   '', 
   false,
   ''),
  
  (nuevo_id, '', 
   '', 
   true,
   ''),
  
  (nuevo_id, '', 
   '', 
   false,
   '');
END $$;
